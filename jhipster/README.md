# JHipster

Prerequisites:

* JDK 8
* Node.js
* Yarn

Then:

```bash
yarn global add yo # to use JHipster Marketplace
yarn global add generator-jhipster
```

## Create app

```bash
mkdir jhipster-demo && cd jhipster-demo
jhipster
```

## Build app

```bash
gradlew
```
